# Exoplanet-Detection-AI
AI-Enabled Detection of Exoplanets from Noisy Astronomical Light Curves

Overview

This project develops an artificial intelligence pipeline for detecting exoplanets from noisy astronomical light curve data. Exoplanets are identified by analyzing periodic decreases in a star's brightness caused by planetary transits.

The proposed system combines signal preprocessing, noise reduction, feature extraction, deep learning classification, and explainable AI techniques to improve detection accuracy and reliability.

Objectives

- Detect exoplanet transit signals from noisy light curve observations.
- Compare multiple denoising techniques for signal enhancement.
- Develop a deep learning model for exoplanet classification.
- Interpret model predictions using explainable AI methods.
- Generate confidence scores for detected exoplanet candidates.

Methodology

1. Data acquisition and preprocessing.
2. Noise reduction using filtering and denoising methods.
3. Extraction of transit-related features.
4. Training of machine learning and deep learning models.
5. Performance evaluation and comparison.
6. Explainability analysis and confidence estimation.

Expected Outcomes

- Explainable predictions for improved scientific interpretation.
- A reproducible AI pipeline for astronomical signal analysis.
